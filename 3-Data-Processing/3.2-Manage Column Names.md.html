3.2 Manage Column Names

Description:

It's common to need to manage how columns names appear in a dataset. This includes specifying which columns should appear in a dataset, which columns should not appear in a dataset, giving columns new names, and controlling how columns are ordered in the dataset.

Example:

# Libraries used here
import pandas as pd
import numpy as np

# Create an empty dataframe
DF = pd.DataFrame()

# Add new columns using lists of values
DF['A'] = [1,2,3]
DF['B'] = ['a','b', 'c']
DF['C'] = ['how',  'about','that']

Out[xx]: 
   A  B      C
0  1  a    how
1  2  b  about
2  3  c   that

SELECT and specify which columns appear in a DataFrame

DF[['A', 'B']]

Out[xx]: 
   A  B
0  1  a
1  2  b
2  3  c

DROP and remove columns. Specify which columns should not appear in a DataFrame

# use .drop(['column_name'], axis = 1)
DF.drop(['B'], axis=1)

Out[xx]:
   A      C
0  1    how
1  2  about
2  3   that

Rename and Alias columns by giving columns new names

# create a dictionary that maps your old columns to new columns
# use .rename(columns = {'OLD':'new'})

new_columns = {'A':'new_a'
              ,'B':'new_b'
              ,'C':'new_c'}

DF.rename(columns = new_columns)

Out[xx]: 
   new_a new_b  new_c
0      1     a    how
1      2     b  about
2      3     c   that

Reorder columns and control how columns are ordered in DataFrame

# let's put the columns in reverse order from what they were before
DF = DF[['C','A','B']]

Out[xx]: 
       C  A  B
0    how  1  a
1  about  2  b
2   that  3  c

Discussion:

While using '.dtypes' to print out column names and their data types is more descriptive, getting a list of column names in 'list' format is useful when you need column names in list format. For example, if you need to add these column names to some other place in your code, having a list of column names in the right format saves you from manually formating quotation marks and commas around each column name, which is a real pain.
