3.3 Conditional Logic Examples

Description:

Here is a collection of conditional logic examples.

Example:

'If Then' Logic Filters

Filtering a DataFrame consists of two steps. First, write a true/false statement. Second, use the true/false statement to filter the DataFrame to cases where the statement = True

Create a True/False statement using a DF column, and return True/False

DF['B'] == 'b'

Out[xx]: 
0    False
1     True
2    False

Use a True/False statement to FILTER a DataFrame.

The basic structure to use is DF[ (your_logic) ]. This Returns a DataFrame where the logic statement = True 

DF[ (DF['B'] == 'b') ]

Out[117]: 
   A  B      C
1  2  b  about

If/Then Logic Examples:

Where column IS equal to a value

DF[( DF['B'] == 'b' )]

Out[103]: 
   A  B      C
1  2  b  about

Where column NOT equal to a value

DF[( DF['B'] != 'b' )]

Out[104]: 
   A  B     C
0  1  a   how
2  3  c  that

Where column is NULL

DF[( DF['has_nulls'].isnull() )]

Where column NOT NULL

DF[( DF['has_nulls'].notnull() )]

Where multiple conditions are all true - AND

# use '&' to seporate your true/false statements
DF[ ( DF['B'] == 'b' )  &  ( DF['A'] == 2 ) ]

Where one or more condition is true - OR

# use '|' to seporate your true/false statements
DF[ ( DF['B'] == 'b' )  |  ( DF['B'] == 'c' ) ]

Where value in column is is in a list of values

# use .isin(['your','list', 'of', 'values'])
# create a true/false condition using a list of values
DF['B'].isin(['a','c'])

# use it to filter a DF
DF[ ( DF['B'].isin(['a','c']) ) ]

Where column NOT IN list of values

# use the ~ before your true/false condition
DF[ ~( DF['B'].isin(['a','c']) ) ]
