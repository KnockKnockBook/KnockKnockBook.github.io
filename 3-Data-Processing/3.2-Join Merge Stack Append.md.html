Combine Multiple DataFrames Together

JOIN or MERGE two dataframes

pd.merge(DF1,DF2
    , left_on  = 'DF1_ID'
    , right_on = 'DF2_ID'
    , how = 'left')

STACK and UNION DataFrames on top of each other

# use pd.concat() and make sure the column names are the same
pd.concat([df1, df2, df3, dfx], ignore_index=True)

Append a new row to a DataFrame

Option 1: Add Rows to a DataFrame with a dictionary

# create dictionary with the column names a new values for each column
new_row = {"A":10, "B":'z', "C":"business","has_nulls": "here here"}

# use .append() to add the row to your DF
DF = DF.append(new_row, ignore_index=True)

Option 2: Add Rows to a DataFrame with a pd.Series

# create pd.Series, and specify the column names with index
new_row_from_series = pd.Series([99, 'x', 'government'], index = ['A', 'B', 'C'])
# use .append() to add the row to your DF
DF.append(new_row_from_series, ignore_index=True)
